# Project Description

A web application that allows users to merge multiple PDF files into a single PDF

Built with a React frontend and a Node.js backend

Supports uploading multiple PDFs at once

Automatically merges PDFs in order and returns a downloadable merged file

Fast, lightweight, and easy to use

Handles file uploads and temporary storage securely

Designed with clean UI and simple workflow

# Tools & Technologies Used
Frontend

React.js

Axios (API requests)

TailwindCSS (for styling)

Backend

Node.js

Express.js

Multer (file upload handling)

pdf-merger-js (PDF merging engine)

CORS (cross-origin support)
